## ✨ Features

| Feature | Description |
|---|---|
| ⏱ **Millisecond Accuracy** | Uses `performance.now()` for sub-millisecond precision |
| 🏁 **Unlimited Lap Recording** | Capture splits with delta time per lap |
| 🟢🔴 **Best / Worst Lap** | Automatically highlights fastest and slowest laps |
| 📱 **Fully Responsive** | Works on desktop, tablet, and mobile |
| ⌨️ **Keyboard Shortcuts** | `Space` = Start/Pause · `L` = Lap · `R` = Reset |
| 🎨 **Animated UI** | Particle canvas background, rotating clock rings, smooth transitions |
| 🔒 **Privacy First** | 100% client-side — no data sent to any server |
| ⚡ **Zero Dependencies** | Pure vanilla HTML/CSS/JS — loads instantly |
| 📴 **Works Offline** | No internet needed after first load |

## 🚀 Getting Started

### Option 1 — Open Directly (Quickest)

```bash
# 1. Clone the repository
git clone https://github.com/YOUR-USERNAME/chronotrack.git

# 2. Navigate into the folder
cd chronotrack

# 3. Open in your browser
# On Windows:
start index.html

# On macOS:
open index.html

# On Linux:
xdg-open index.html
```

That's it — no installs, no build step, no server required.

---

### Option 2 — Using VS Code Live Server

1. Open the project folder in **VS Code**
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **"Open with Live Server"**
4. Browser opens at `http://127.0.0.1:5500`

---

### Option 3 — Using Python (Local Server)

```bash
# Python 3
cd chronotrack
python -m http.server 8080

# Then open in browser:
# http://localhost:8080
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Start / Pause the stopwatch |
| `L` | Record a lap (while running) |
| `R` | Reset the stopwatch |

> Shortcuts are disabled when typing in the contact form.
